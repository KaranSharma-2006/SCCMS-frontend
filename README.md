Smart City Complaint Management System (SCCMS)
Overview

SCCMS is a web-based platform built to simplify how citizens report civic issues and how authorities handle them. The goal is to reduce delays, improve transparency, and make complaint tracking straightforward for both users and admins.

This project focuses on real-world usability — allowing users to submit complaints easily while giving administrators a structured system to manage and resolve them efficiently.

What This Project Does
Allows users to register and log in securely
Lets citizens submit complaints with relevant details
Provides status tracking so users know what’s happening
Gives admins a dashboard to view, manage, and update complaints
Organizes complaints into categories for better handling
How It Works

Users create an account and submit complaints through the frontend interface. These complaints are stored in the database and processed through the backend API. Admins can then review, update, and mark complaints as resolved.

Tech Used

Frontend: React.js
Backend: FastAPI / Node.js
Database: MongoDB / MySQL

Why This Project

City complaint systems are often slow and unclear. This project aims to make the process faster, more transparent, and easier to use with a simple digital solution.
